# CAPEGEMINI_TRAINNING_JAVA_B3

## 📌 About This Repository
This repository contains Java training practice programs completed during Capgemini Java Training.

It includes basic to intermediate level concepts of Java programming along with hands-on practice examples.

- for loop
- while loop

- Pattern programs
- Problem solving programs

### 🔹 Arrays
- 1D Arrays
- 2D Arrays

### 🔹 Strings

### 🔹 OOP Concepts

- Constructors

- Encapsulation

- Method Overriding
- Abstraction
- Interfaces
